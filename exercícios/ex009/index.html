Listas Ordenadas

As listas ordenadas são aquelas em que a ordem dos elementos importa. Aqui estão os diferentes tipos de listas ordenadas:

Tipo 1: lista numerada.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar a roupa
  6. Ir para a faculdade

Tipo a: lista com letras minúsculas.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar a roupa
  6. Ir para a faculdade

Tipo A: lista com letras maiúsculas.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar a roupa
  6. Ir para a faculdade

Tipo i: lista com números romanos minúsculos.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar a roupa
  6. Ir para a faculdade

Tipo I: lista com números romanos maiúsculos.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Lavar a roupa
  6. Ir para a faculdade
